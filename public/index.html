How to Create Google OAuth Credentials

Copy this exact URI into Google Cloud Console under Authorized redirect URIs. The protocol, domain, port, path and trailing slash must match exactly.

  1. Step 1 - Create a Google Cloud Project

    Open Google Cloud Console, sign in with the Gmail account used for Smart Fin 365, create a project named "Smart Fin 365", and select it.

    Google Cloud Console
  2. Step 2 - Enable Google APIs

    Open APIs & Services - Library. Enable Google Drive API, Google Sheets API, and Gmail API only if the application sends Gmail messages.

    API Library
  3. Step 3 - Configure Google OAuth Branding

    Open Google Auth Platform, enter "Smart Fin 365" as the application name, add support/developer emails, upload the logo, and save.

    OAuth BrandingGoogle branding guide
  4. Step 4 - Configure the Audience

    Select External for personal Gmail accounts. Use Internal only for Google Workspace. Keep Testing during setup and add the Smart Fin 365 Gmail address as a test user.

    OAuth AudienceAudience guide
  5. Step 5 - Configure Data Access

    Add only required scopes: drive.file, spreadsheets, openid, email, profile, and gmail.send only when Gmail sending is enabled.

    OAuth ScopesData access guide
  6. Step 6 - Create the OAuth Client

    Open Google Auth Platform - Clients, create a Web application client, and name it "Smart Fin 365 Web Client".

    OAuth ClientsOAuth client guide
  7. Step 7 - Add Authorized JavaScript Origins

    For local use enter http://localhost:3333. For production enter the exact HTTPS domain. Do not include a page path.

  8. Step 8 - Add Authorized Redirect URIs

    Use the exact Authorized Redirect URI displayed above. Local is normally http://localhost:3333/oauth2callback. Production is generated from the configured public application URL.

  9. Step 9 - Copy the Credentials

    Copy the complete OAuth Client ID and Client Secret into Smart Fin 365, click Save Drive Setup, Connect Gmail, approve permissions, and test Upload Excel Now.
